# GrowinCamera Authentication App is a modern web-based application packaged as an Android app
using Trusted Web Activity (TWA). It allows camera-based interactions directly through
the browser environment, optimized for environments without built-in webcams and
supporting virtual camera sources.

The app is built with React and TypeScript, deployed with Docker, and uses a secure
HTTPS connection. No personal data is collected or stored.g-Business
